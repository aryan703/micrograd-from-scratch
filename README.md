# micrograd-from-scratch

Building Karpathy's micrograd autograd engine from scratch.

Part of my [14-week ML learning plan](https://github.com/aryan703/aryank) — Week 1.

## Status
🟡 In progress — Week 1 of plan (Jun 1-6, 2026)

## What this is
A tiny autograd engine that implements reverse-mode auto-differentiation from first principles. ~150 lines of pure Python, no PyTorch.

## Goals
- [ ] Implement `Value` class with `+`, `*`, `tanh`
- [ ] Implement backward pass via topological sort
- [ ] Build `Neuron`, `Layer`, `MLP` classes on top
- [ ] Train a tiny MLP on the moons dataset
